# Archived Documentation

This directory contains historical and maintainer-oriented documentation that is
kept for context, not as implementation authority.

For active contributor and operator workflows, prefer:

- `README.md` (root onboarding)
- `backend/README.md`
- `IOS/README.md`
- `docs/operations/`

## What Lives Here

- `ios-history/`
  - historical iOS migration/context material
  - includes archived DAT context such as `Wearables_DAT_SDK.md`
- `maintainer-roadmaps/`
  - planning and implementation-roadmap documents for maintainers
  - these docs are public but not product/support commitments

## Policy

- Archived docs are reference-only.
- Do not treat archived docs as the source of truth over active runtime code and
  active operations docs.
- Onboarding and quickstart flows should not require archived docs.
